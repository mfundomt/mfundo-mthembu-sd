## 👣 FootLook — Zero-Interaction API Observability

A shadow service that passively mirrors and captures API traffic flowing through a system without modifying the target API behavior.

### Highlights
- Passive traffic observation architecture
- Non-invasive middleware capture strategy
- Live traffic feed concept for debugging and analysis

### Stack
- C#, .NET 8, ASP.NET Core Middleware
- SQL Server + in-memory cache
- SignalR for live updates
- Angular dashboard
