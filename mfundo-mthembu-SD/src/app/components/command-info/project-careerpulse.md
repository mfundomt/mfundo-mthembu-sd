## 📈 CareerPulse — Intelligent Web Crawler + Newsletter System

An automation pipeline that monitors inbound newsletters, classifies relevance, and delivers concise daily summaries.

### Highlights
- Automated content intake and filtering
- Relevance scoring for career-focused topics
- Daily digest workflow and notification delivery

### Stack
- n8n orchestration
- Scrapy
- Email IMAP/API integration
- Push notification API
- SQL Server for history and deduplication
