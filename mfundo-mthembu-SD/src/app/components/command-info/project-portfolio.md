## 🔧 Personal Portfolio (This Site)

A terminal-themed portfolio experience where users navigate sections via commands and interactive modals.

- Command-based navigation
- Recruiter/Developer mode switching
- Markdown-driven content rendering
- SSR-first architecture for speed and SEO

### Stack
- Angular 21 + SSR
- TypeScript
- Angular Material
- marked.js
- Express 5
